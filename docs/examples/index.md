# Bot Examples

Complete, working bot examples showcasing EasyLang features.

## Available Examples

### [Basic Bot](/EasyLang/examples/basic-bot)
Simple command bot with reactions and embeds. Perfect for beginners.

**Features:** Commands, embeds, reactions, server info

### [Music Bot](/EasyLang/examples/music-bot)
Full-featured music bot with queue management and controls.

**Features:** Voice, YouTube playback, queue system, controls

### [Moderation Bot](/EasyLang/examples/moderation-bot)
Complete moderation system with AutoMod and logging.

**Features:** Kick, ban, timeout, AutoMod, audit logs

### [Utility Bot](/EasyLang/examples/utility-bot)
Useful utility commands like polls, reminders, and info.

**Features:** Polls, invites, server stats, user info

### [Event Bot](/EasyLang/examples/event-bot)
Scheduled events and reminders system.

**Features:** Scheduled events, reminders, tasks

### [Advanced Bot](/EasyLang/examples/advanced-bot)
Comprehensive bot combining all features.

**Features:** Everything combined with proper structure
