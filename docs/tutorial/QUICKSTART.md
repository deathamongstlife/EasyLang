# Interactive Tutorial Quick Start Guide

Get started with the EasyLang interactive tutorial in 5 minutes!

## What You'll Learn

The interactive tutorial teaches you to build Discord bots through 10 hands-on lessons:

1. **Variables** - Store and work with data
2. **Functions** - Create reusable code
3. **Messages** - Reply to Discord messages
4. **Embeds** - Create rich embedded messages
5. **Conditions** - Make decisions in code
6. **Loops** - Repeat actions efficiently
7. **Reactions** - Add emoji reactions
8. **Events** - Listen to Discord events
9. **Buttons** - Create interactive buttons
10. **Complete Bot** - Build a full bot

**Time Required:** 30-60 minutes for all lessons
**Prerequisites:** None! Absolute beginners welcome.

## Troubleshooting

### Tutorial won't load
- Check JavaScript is enabled
- Try refreshing the page
- Clear browser cache
- Try a different browser

### Code won't run
- Click the "â–¶ Run Code" button
- Try Ctrl+Enter keyboard shortcut
- Check for console errors (F12)

### Progress lost
- Check browser localStorage is enabled
- Try the same browser/device
- Progress doesn't sync across devices

### Validation failing
- Read error message carefully
- Check hint for guidance
- Try the exact syntax suggested
- Use solution if stuck

### Mobile issues
- Rotate to landscape for more space
